# 🎉 友人向け セットアップガイド

## これは何？

世界5ヶ国の大学・研究機関から毎日自動で最新の学術論文を集めて、
**AIが日本語で分析・解説してくれる**ニュースサイトです！

複数の友人と一緒に使えて、URLをシェアするだけでアクセスできます。

---

## ⚡ すぐに始める（3ステップ）

### ステップ1️⃣: サイトを開く

友人から受け取った URL を開く

```
https://global-research.example.com
```

✨ これだけ！セットアップ不要です

### ステップ2️⃣: 論文を眺める

- 📄 カードをクリックして論文の詳細を見る
- 🤖 AI分析で日本語解説を読む
- 🔗 arXiv のリンクで全文を確認

### ステップ3️⃣: 更新リクエスト（オプション）

すぐに新しい論文を取得したい場合：

```
「今すぐ更新」ボタンをクリック
```

バックグラウンドで自動実行（10秒後に反映）

---

## 🤖 AI分析について

各論文には以下が自動生成されます：

### 【分析】
- この研究の学術的意義
- 既存研究との違い
- 重要なポイント

### 【発展性】
- 今後の応用可能性
- 関連分野への影響
- 実用化の見込み

※ AI 分析は自動生成のため、確認は arXiv の原文も見てください

## 💾 データについて

### 自動更新
毎日 09:00（日本時間）に自動更新されます

### 何が保存される？
```
✓ 論文タイトル
✓ 著者情報
✓ 概要
✓ 公開日
✓ arXiv リンク
✓ AI 分析（日本語）
```

### プライベート？
このサイトはローカルホストで実行されるため：
- 検索履歴は記録されない
- ユーザー情報は保存されない
- 完全プライベート

## 🔗 API（開発者向け）

### JavaScript で使う例

```javascript
// 全論文を取得
fetch('https://global-research.example.com/api/papers')
  .then(r => r.json())
  .then(d => console.log(d))

// 統計情報を取得
fetch('https://global-research.example.com/api/stats')
  .then(r => r.json())
  .then(d => console.log(d))

// キーワード検索
fetch('https://global-research.example.com/api/papers/search?keyword=AI')
  .then(r => r.json())
  .then(d => console.log(d))
```

### Python で使う例

```python
import requests

# 全論文を取得
response = requests.get('https://global-research.example.com/api/papers')
data = response.json()
print(f"総論文数: {data['total_papers']}")

# キーワード検索
response = requests.get(
    'https://global-research.example.com/api/papers/search',
    params={'keyword': 'AI'}
)
results = response.json()
print(f"検索結果: {len(results['results'])}件")
```

### curl で使う例

```bash
# 全論文を取得
curl https://global-research.example.com/api/papers

# 特定国の論文を取得
curl https://global-research.example.com/api/papers?country=JP

# API 健全性チェック
curl https://global-research.example.com/health
```

## 🐛 トラブル

### 論文が表示されない

```
原因: API サーバーが起動していない
対策: サイト所有者に連絡
```

### データが古い

```
原因: 自動更新がまだ実行されていない
対策: 「今すぐ更新」ボタンをクリック
```

### モーダルが開かない

```
原因: JavaScript が無効
対策: ブラウザのコンソールを確認
```

### AI 分析が表示されない

```
原因: API キーの設定ミス
対策: サイト所有者に連絡
```
